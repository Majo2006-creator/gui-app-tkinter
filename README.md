# Aplicación GUI Básica con Tkinter

## Descripción
Esta aplicación fue desarrollada en Python utilizando la librería Tkinter. Permite a los usuarios ingresar información a través de un campo de texto y visualizarla en una lista dentro de una interfaz gráfica.

## Objetivo
Desarrollar una interfaz gráfica básica que permita interactuar con datos mediante componentes GUI.

## Componentes utilizados

- Ventana principal
- Etiquetas (Label)
- Campo de texto (Entry)
- Botones (Button)
- Lista de datos (Listbox)

## Funcionalidades

✔ Agregar información mediante un campo de texto  
✔ Mostrar la información en una lista  
✔ Eliminar o limpiar datos  
✔ Responder a eventos de clic en botones

## Tecnologías utilizadas

- Python
- Tkinter
